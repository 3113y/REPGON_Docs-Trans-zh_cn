# Class "AnimationLayer"

已加载ANM2文件中动画某一层的缓存数据。由所有使用相同ANM2的[Sprites](Sprite.md)共享，且不可修改。

可通过[AnimationData:GetLayer()](AnimationData.md#getlayer)获取。.
